# Dash Finish Line

```
git clone https://github.com/AlgorithmHub/finishline.git
cd finishline
python3 server.py
```

To create a new viz block:

```
cd finishline
cp -r plugins/HelloWorld plugins/MyNewBlock
vi plugins/MyNewBlock
```

State of project is proof of concept.
